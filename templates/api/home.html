PrivLink

API

PrivLink was made with the intention to be integrated seamlessly into other apps and services. The API is request access only to prevent potential spam - this would not take away the anonimity aspect of the service but it would mean all requests would have to be sent via a API key.

To get access or if you have any questions, feel free to mail us or contact me at our Discord server.
